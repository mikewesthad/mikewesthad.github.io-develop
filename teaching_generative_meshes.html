Generative Meshes (2013)

openFrameworks is an open source C++ toolkit to make creating things with code easier. I wrote a tutorial, hosted on the openFrameworks site here, on the basics of using code to generate 3D shapes. It is aimed at beginners: those who understand variables, loops and how to create projects in openFrameworks.

The lesson walks through what a 3D object is in code; how you can take an image and use it to generate a 3D object; and then once you've generated a 3D object, what can you do with it. It starts with an image from the Hubble telescope (left) and ends with turning it into a 3D object (right) that can then be animated and manipulated.
