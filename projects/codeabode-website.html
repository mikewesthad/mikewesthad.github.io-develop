Overview

In collaboration with Rex Twedt, I designed and implemented a modern, responsive website for CodeAbode, a Chicago-based, not-for-profit code school.

A live version of the design is hosted here.

The site was built so that it could be easily maintained and updated when the schedule changes or when new courses are offered. It was built with a modern set of tools, including Bootstrap, Gulp, SASS, HTML templating and PHP. The source code can be found here.
